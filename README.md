# obstacle-avoidance-robot
A fully autonomous robot that navigates on its own using an  ultrasonic sensor and servo motor. No remote. No control.  Just place it on the ground and it moves, detects, scans,  decides, and avoids — all by itself.  Built by **Sarvesh Kumar** — 2nd Year B.Sc.(H) Electronics Student   Completely self-taught. No mentor. Just curiosity.
